# Lexical Analysis - Lab Task

## Task Description
The objective is to tokenize a given input string and identify the type of each token (Keyword, Identifier, Operator, Number, or Symbol).

**Input String:**
`int sum = 10; a + b = 20;`

## Token Identification- Output
| Token | Type |
|-------|------|
| int   | Keyword |
| sum   | Identifier |
| =     | Operator |
| 10    | Number |
| ;     | Symbol |
| a     | Identifier |
| +     | Operator |
| b     | Identifier |
| =     | Operator |
| 20    | Number |
| ;     | Symbol |
